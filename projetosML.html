Previsão de Fraude

Neste projeto, criei um modelo de Machine Learning para previnir fraudes em transações de cartão de crédito. Após análise e tratamento dos dados, modelei os dados utilizando o StandardScaler para ajustar as escalas e o Imbalanced-learn para balancear o dataset. Em seguida, apliquei varios algoritmos de classificação e, após selecionar o modelo com melhor desempenho, utilizei o GridSearchCV para otimizar os hiperparâmetros e aperfeiçoar o modelo. Prevendo consumo e vendas

Neste projeto, analisei um conjunto de dados sobre o consumo de cerveja em um determinado estabelecimento, levando em consideração variáveis como temperatura, época do ano, entre outros. Após realizar análises preliminares dos dados e suas variáveis, utilizei regressão linear para realizar previsões. Por fim, avaliei os resultados com base nas métricas de desempenho. Prevendo vendas conforme investimento

Neste projeto, a partir dos dados de uma empresa, que contem os valores investidos em marketing e o valor das vendas, desenvolvi um modelo para prever as vendas com base nos investimentos. Após analisar e tratar os dados, utilizei a regressão linear para criar o modelo e avaliei seu desempenho. Clique e confira !

Prevendo perda de clientes

Neste projeto, analisei dados de clientes e suas características em relação a determinada empresa, visando prever quais clientes podiam vir a se desvincular. Após o tratamento e balanceamento dos dados, utilizei os modelos Bernoulli de Naive Bayes, Knn e árvore de decisão para realizar as previsões. Por fim, avaliamos o desempenho dos modelos por meio de métricas de avaliação.

InvestPartner:
Seu parceiro de Investimentos

InvestPartner é um assistente pessoal financeiro que auxilia o usuário respondendo questões relacionadas às ações da B3. Utilizando a API YFinance, o agente busca dados das ações para fornecer informações como cotação atual, histórico de cotações, detalhes técnicos e muito mais. Para isso, utilizei a biblioteca da OpenAI, o Streamlit para desenvolver a interface e o Hugging Face para realizar o deploy.
